SeBlockly > web-based visual programming editor for Selenium

Welcome to SeBlockly Playground!

SeBlockly Playground is a place where you can test all available SeBlockly blocks.
Feel free to create and export your own selenium code for a java file and after finishing the build process run your own web automation using selenium.
Have fun with blocks and create your own automation scripts!
