How To

  1. Write your tests using drag-and-drop feature above
  2. Click on the button Save Selenium Code to save your code.
  3. Open Eclipse IDE (or another if you want) and create a class with same name of Selenium Code
  4. Organize the imports with command crtl + shift + O
  5. Run your tests

About

SeBlockly Playground is a place where you can test all available SeBlockly blocks. Feel free to create and export your own selenium code for a java file and after finishing the build process run your own web automation using selenium.

If you are new with automation, this is a best start point to learn and improve your skills! Have fun with blocks and create your own automation scripts!
